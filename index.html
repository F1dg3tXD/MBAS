Live note TTS (browser)

Uses your browser’s text‑to‑speech to approximate DekTalk timing. Exact sound in Moonbase Alpha will differ.

Core DekTalk codes

  • [:rate x] – speed (e.g. [:rate 200])
  • [:dv x] – distortion / voice warp
  • [:nh] – whisper / harsh mode
  • [text<dur,pitch>] – phoneme note

Tips

  • 128 char hard limit per message
  • Punctuation is spoken literally
  • Weird spacing + caps = new sounds
  • Avoid EST PST CST MST spam (can crash)
